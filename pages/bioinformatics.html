Bioinformatics Projects

  • MosaicsPerl

    A perl wrapper for the MOSAiCS CHIP-SEQ R package. Makes a default analysis simple, while also supporting all of the non-default options. Uses the Statistics::R perl module. MOSAiCS: Chung D, Kuan PF and Keles S (2012). mosaics: MOSAiCS (MOdel-based one and two Sample Analysis and Inference for ChIP-Seq). R package version 1.12.0

  • Easy-Variant

    A simplified variant/SNP caller implemented in Perl, using Moose Object library. Calls SNPs and Indels, can be configured with a variety of options such as setting a minimum depth, or ignoring certain ranges of the genome. The genome package stores all sequence information in MySQL.

  • perlpipe

    A collection of various scripts and pipelines. The perl directory contains a large majority of the useful stuff. There are pipelines scripts for bowtie2 alignment with samtools rocksort, and a MOSAiCS pipelines which uses the MosaicsPerl.pm module.
